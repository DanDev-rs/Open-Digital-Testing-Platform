Задуманы 2 версии программы: ученическая версия (studED) и учительская версия (teachED)

Ученическая версия:
	Подключения к компьютеру учителя
	Скачивание тестов (зашифрованный файл + скрытый ключ)
	Прохождение теста в реальном времени/на время
	Просмотр результатов из базы данных
	
Учительская версия:
	Создание тестов из нескольких вопросов:
		*С одним ответом (ручная проверка)
		*С выбором одного ответа из нескольких вариантов (до 6)
		*С выбором нескольких правильных ответов из группы возможных (до 8)
	Просмотр базы данных с результатами
	Работает как сервер с тестами
	Ученики могут подключиться с помощью уникального пароля
	Пароль, необходимый для подключения, создаётся во время первого запуска
	Данные хранятся в файле usersettings.mmf, который создаётся во время первого запуска
